# Runtime Runbook (Agent Onboarding)

This document is the fastest way to run and validate the current fundamentals pipeline.

## 1) Prerequisites

- Python environment with project dependencies installed.
- Valid IBKR login flow through the browser session helper.
- `data/ib_products.csv` populated (use product scraper if missing).

## 2) Standard Execution Flow

1. Validate/login session:
- `python -m pystocks.session`

2. Refresh ETF universe (if needed):
- `python -m pystocks.product_scraper`

3. Run fundamentals scrape:
- `python -m pystocks.fundamentals --force=True --limit=50`

4. Optional full normalized rebuild from CAS events:
- `python -m pystocks.cli backfill_fundamentals_normalized`

5. Refresh DuckDB views only:
- `python -m pystocks.cli refresh_fundamentals_views`

## 3) Immediate Outputs To Check

- Telemetry:
  - `data/research/fundamentals_run_telemetry_latest.json`
- Manifest:
  - `data/fundamentals/events.db`
- Query layer:
  - `data/fundamentals/fundamentals.duckdb`

## 4) Fast Validation Queries (DuckDB)

- Coverage by endpoint:
```sql
SELECT * FROM endpoint_catalog ORDER BY endpoint;
```

- Factor feature health:
```sql
SELECT * FROM factor_features_catalog ORDER BY feature_group, feature_name LIMIT 200;
```

- Series catalogs:
```sql
SELECT * FROM price_chart_series_catalog ORDER BY conid LIMIT 50;
SELECT * FROM sentiment_search_series_catalog ORDER BY conid LIMIT 50;
SELECT * FROM ownership_trade_log_series_catalog ORDER BY conid LIMIT 50;
SELECT * FROM dividends_events_series_catalog ORDER BY conid LIMIT 50;
```

- Point-in-time factor panel sample:
```sql
SELECT *
FROM factor_panel_long_daily
ORDER BY trade_date DESC, conid
LIMIT 200;
```

## 5) Operational Notes

- `morningstar` effective date uses the latest publish date found in payload; if absent, it falls back to `as_of_date`.
- `sentiment_search` price/OHLC fields are stripped before storage.
- `ownership_trade_log` excludes `NO CHANGE` actions.
- `dividends` and `ownership` keep full raw payloads in CAS even when analytics excludes embedded price series.
