# IBKR Portal API Reference

The project uses direct JSON interception via the IBKR Portal Proxy. These endpoints are available within a logged-in IBKR web session.

## Base URL
All fundamental endpoints are prefixed with:
`https://www.interactivebrokers.ie/tws.proxy/fundamentals/`

## 1. Core Fundamental Endpoints

### `mf_holdings/{conId}`
**Returns:** Detailed portfolio breakdown.
*   `allocation_self`: Asset class weights (Equity, Cash, Bond).
*   `top_10`: Top 10 holdings with `conids` and `assets_pct`.
*   `industry`: Sector weights (e.g., Technology, Healthcare).
*   `investor_country`: Geographic exposure weights.

### `mf_profile_and_fees/{conId}`
**Returns:** Fund profile, expenses, and PDF report metadata.
*   `fund_and_profile`: Key metadata (Domicile, TER, Benchmark).
*   `reports`: Annual and Prospectus report summaries with `as_of_date`.
*   `expenses_allocation`: Management vs. Non-management expenses.

### `mf_ratios_fundamentals/{conId}`
**Returns:** Comprehensive set of valuation and growth ratios.
*   `ratios`: Price/Sales, Price/Earnings, Dividend Yield, etc.
*   `zscore`: Composite Z-scores for valuation metrics.

### `landing/{conId}?widgets={...}`
**Returns:** A bulk fetch of multiple data "widgets".
*   **Common Widgets**: `objective`, `mstar`, `lipper_ratings`, `mf_key_ratios`, `risk_and_statistics`, `holdings`, `performance_and_peers`, `keyProfile`, `ownership`, `dividends`, `mf_esg`.

## 2. Analyst & Performance Endpoints

### `mf_lip_ratings/{conId}`
**Returns:** Lipper ratings across various time horizons (3Y, 5Y, 10Y, Overall).
*   Categories: Total Return, Consistent Return, Preservation, Expense.

### `mf_performance/{conId}`
**Returns:** Cumulative and annualized performance vs benchmark.
*   Includes `min`, `max`, and `avg` for the category.

### `mf_risks_stats/{conId}?period={...}`
**Returns:** Historical risk measures (Sharpe, Std Dev, VaR, Max Drawdown).
*   `period` options: `6M`, `1Y`, `3Y`, `5Y`, `10Y`.

## 3. Alternative & ESG Endpoints

### `ownership/{conId}`
**Returns:** Institutional and insider ownership data.
*   `ownership_history`: Time-series of institutional/strategic holdings.
*   `owners_types`: Breakdown by manager type (Inv. Managers, Brokerage, etc.).

### `dividends/{conId}`
**Returns:** Historical dividend payments and industry comparison.
*   Includes ex-dates, pay dates, and amounts.

### `impact/esg/{conId}`
**Returns:** Environmental, Social, and Governance scores from Refinitiv/Lipper.

### `sma/request?type={search|tick|high_low}&conid={conId}`
**Returns:** Social Media Sentiment data.
*   `type=search`: Time-series of sentiment scores, buzz, and volume.
*   `type=tick`: Real-time sentiment snapshot.

## 4. Morningstar Endpoint
*Note: Uses a different base path.*
`https://www.interactivebrokers.ie/tws.proxy/mstar/fund/detail?conid={conId}`
**Returns:** Morningstar Medalist and Star ratings, Process/People/Parent pillars.
