# Operational Checklist (Current)

Use this checklist to validate a full run from ingestion through analysis.

## 1) Post-Ingestion Checks
1. Telemetry exists:
- `data/research/fundamentals_run_telemetry_latest.json`

2. Endpoint events exist:
```sql
SELECT COUNT(*) AS n_events FROM endpoint_events_all;
```

3. Factor features exist:
```sql
SELECT COUNT(*) AS n_factor_rows FROM factor_features_all;
```

4. Series stores are visible:
```sql
SELECT COUNT(*) AS n_price_rows FROM price_chart_series_all;
SELECT COUNT(*) AS n_sentiment_rows FROM sentiment_search_series_all;
SELECT COUNT(*) AS n_ownership_rows FROM ownership_trade_log_series_all;
SELECT COUNT(*) AS n_dividend_rows FROM dividends_events_series_all;
```

## 2) Data Integrity Checks
- Ownership `NO CHANGE` rows should be absent:
```sql
SELECT COUNT(*) AS no_change_rows
FROM ownership_trade_log_series_all
WHERE upper(action) = 'NO CHANGE';
```

- Factor lineage should join back to endpoint events:
```sql
SELECT
  COUNT(*) AS factor_rows,
  SUM(CASE WHEN e.event_id IS NULL THEN 1 ELSE 0 END) AS missing_event_join,
  SUM(CASE WHEN e.payload_hash IS DISTINCT FROM f.payload_hash THEN 1 ELSE 0 END) AS payload_hash_mismatch
FROM factor_features_all f
LEFT JOIN endpoint_events_all e
  ON f.endpoint_event_id = e.event_id;
```

- Sentiment stripped-price fields should not appear:
```sql
SELECT column_name
FROM information_schema.columns
WHERE table_name = 'sentiment_search_series_all'
  AND column_name IN ('price','open','high','low','close','price_change','price_change_p');
```

- Price date integrity checks:
```sql
SELECT COUNT(*) AS weekend_rows
FROM price_chart_series_all
WHERE EXTRACT(ISODOW FROM TRY_CAST(trade_date AS DATE)) IN (6,7);

SELECT COUNT(*) AS date_mismatch_rows
FROM price_chart_series_all
WHERE timestamp_ms IS NOT NULL
  AND CAST(TO_TIMESTAMP(timestamp_ms/1000.0) AS DATE)
      IS DISTINCT FROM TRY_CAST(trade_date AS DATE);
```

After `python -m pystocks.cli run_pipeline`:

```sql
SELECT COUNT(*) AS n_clean_rows FROM price_chart_series_clean_all;
SELECT COUNT(*) AS n_returns_rows FROM returns_daily_clean;
SELECT COUNT(*) AS n_eligible FROM price_quality_catalog WHERE eligible = TRUE;
```

## 4) If You Need a Clean Rebuild
1. Remove existing data directories you want to reset (for example `data/fundamentals/`, `data/prices/`, `data/dividends/`).
2. Re-run ingestion:
- `python -m pystocks.cli scrape_products`
- `python -m pystocks.cli scrape_fundamentals --limit 100`
3. Refresh views:
- `python -m pystocks.cli refresh_fundamentals_views`
4. Re-run tail pipeline:
- `python -m pystocks.cli run_pipeline`
